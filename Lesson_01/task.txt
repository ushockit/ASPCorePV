Создайте класс Singleton UsersStorage внутри которого будет список объектов Person
Реализуйте маршрутизацию для:
- / - главная страница на которой расположен список всех пользователей в виде ul списка. Возле каждого пользователя сделайте ссылку, которая будет вести по пути /userInfo?id=...
- /userInfo?id=... - маршрут, который отображает информацию о выбранном пользователе. Если в списке не существует пользователь с полученным id, то в ответ отправляйте ошибку 400


Используйте StringBuilder для формирования списка пользователей ul на сервере
